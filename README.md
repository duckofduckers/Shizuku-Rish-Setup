## Overview

This repository provides a **safe and consistent installer** for setting up **rish with Shizuku** inside **Termux**.

The installer handles:
- Correct file placement
- Permission setup
- Basic validation to prevent silent failures

Using the installer is the **recommended and supported** installation method.

---

## Installation

Run the installer directly in Termux:

```sh
bash <(curl -fsSL https://github.com/duckofduckers/Shizuku-Rish-Setup/raw/main/Installer)
```

---

# Uninstallation

To remove all files installed by the installer, run the uninstaller:

```sh
bash <(curl -fsSL https://github.com/duckofduckers/Shizuku-Rish-Setup/raw/main/Uninstaller)
```
